Lecture 4: The Basics of Counting & Permutations

1. The Counting Rules

  • Sum Rule (Mutually Exclusive): If a task can be done in n1 ways or n2 ways, total ways = n1 + n2.
  • Product Rule (Sequential): If a procedure has two tasks done in n1 and n2 ways, total ways = n1 × n2.
  • Subtraction Rule (Inclusion-Exclusion): Used when tasks overlap. Total = n1 + n2 - (number of common ways).
  • Division Rule: Used when items are grouped in indistinguishable ways.

2. Permutations

An arrangement where order matters.

  • Without Repetition: P(n, r) = n! / (n-r)!
  • With Repetition: nr
  • With Indistinguishable Objects: n! / (n1! × n2! × ... × nk!)
