Lecture 6: Mathematical Induction

1. Introduction: The Ladder Analogy

Mathematical induction is a technique used to prove that a propositional function P(n) is true for all positive integers n. Imagine an infinite ladder:

  • Step 1: We can reach the first rung of the ladder.
  • Step 2: If we can reach a particular rung (k), then we can reach the next rung (k + 1).

From (1), we reach rung 1. By applying (2), we reach rung 2. Applying (2) again reaches rung 3. We can apply this rule any number of times to reach any rung, no matter how high up.

2. Principle of Mathematical Induction

To prove that P(n) is true for all positive integers n, we complete these two steps:

  • Basis Step: Show that P(1) is true.
  • Inductive Step: Show that P(k) → P(k+1) is true for all positive integers k.

To complete the inductive step, assume the Inductive Hypothesis that P(k) holds for an arbitrary integer k, then show that P(k+1) must be true.

3. The Domino Analogy

Think of an infinite sequence of standing dominoes:

  • We know the first domino is knocked down (Basis Step: P(1) is true).
  • If the kth domino is knocked over, it knocks over the (k+1)th domino (Inductive Step: P(k) → P(k+1) is true).

Hence, all dominoes are eventually knocked over.

4. Proving a Summation Formula

Example: Show that ∑i=1n i = n(n+1)/2

  • Basis Step: P(1) is true since 1(1+1)/2 = 1.
  • Inductive Step: Assume true for P(k). The hypothesis is ∑i=1k i = k(k+1)/2.
  • Under this assumption, show P(k+1):
    1 + 2 + ... + k + (k+1) = [k(k+1)/2] + (k+1)
    = [k(k+1) + 2(k+1)] / 2
    = (k+1)(k+2) / 2

5. Conjecturing and Proving

Example: Find a formula for the sum of the first n positive odd integers.

Observation: 1=1, 1+3=4, 1+3+5=9, 1+3+5+7=16. Conjecture: sum = n2.

  • Basis Step: P(1) is true as 12 = 1.
  • Inductive Step: Assume 1+3+5+...+(2k-1) = k2.
  • Then, 1+3+5+...+(2k-1) + (2k+1) = k2 + (2k+1) = (k+1)2.

6. Proving Inequalities

Case 1: n < 2n

  • Basis Step: 1 < 21 (True).
  • Inductive Step: Assume k < 2k. Then k+1 < 2k + 1 ≤ 2k + 2k = 2k+1.

Case 2: 2n < n! for n ≥ 4

  • Basis Step: P(4) is true since 16 < 24.
  • Inductive Step: Assume 2k < k!. Then 2k+1 = 2 × 2k < 2 × k! < (k+1) × k! = (k+1)!.

7. Proving Divisibility

Example: Prove n3 - n is divisible by 3.

  • Basis Step: 13 - 1 = 0, which is divisible by 3.
  • Inductive Step: Assume k3 - k is divisible by 3.
  • For (k+1): (k+1)3 - (k+1) = (k3 + 3k2 + 3k + 1) - (k+1) = (k3 - k) + 3(k2 + k).
  • Since (k3 - k) is divisible by 3 (hypothesis) and 3(k2+k) is divisible by 3, the sum is divisible by 3.

8. Tiling Checkerboards

Problem: Show that every 2n × 2n checkerboard with one square removed can be tiled using right triominoes (L-shaped tiles covering 3 squares).

  • Basis Step: P(1) is true.
  • Inductive Step: Split the board into four quadrants. Remove one square from one quadrant. The other three quadrants can be "corrected" by placing one triomino at the center, leaving three squares each missing a corner. By the inductive hypothesis, these can all be tiled.
