Lecture 9: Recursion

1. Recursive Definitions

Solving a problem by breaking it down into smaller subproblems of the same exact form.

  • Basis Step: Specify the function's value at the smallest input (the stopping condition).
  • Recursive Step: Provide a rule for finding its value at larger integers based on smaller ones.

2. Famous Recursive Algorithms

  • GCD (Greatest Common Divisor): gcd(a, b) = gcd(b mod a, a).
  • Binary Search: Halving an sorted array to find a target integer.
  • Tower of Hanoi: Moving disks between pegs using smaller sub-moves.
