Lecture 7: Graph Theory & Search Algorithms

1. Graph Components

  • Vertices (V): The nodes or points in the graph.
  • Edges (E): The connections between vertices. Can be Directed (one-way) or Undirected (two-way).
  • Simple Graph: No multiple edges between the same two nodes, and no loops.
  • Multigraph: Allows multiple edges between the same nodes.

2. Graph Traversal

  • Breadth-First Search (BFS): Explores level-by-level using a Queue. Good for finding the shortest path in unweighted graphs.
  • Depth-First Search (DFS): Explores as deep as possible along a branch before backtracking, using a Stack or recursion.
