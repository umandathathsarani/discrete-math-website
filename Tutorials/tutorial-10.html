Question 1: Algorithm Definition

What is the fundamental definition of an Algorithm?

🪄 Lumos! Reveal Answer

Answer:
It is a step-by-step procedure for solving a specific problem, acting much like a set of instructions or a recipe.

Question 2: Measuring Efficiency

Why do computer scientists avoid using actual time in seconds to measure the speed of an algorithm?

🪄 Lumos! Reveal Answer

Answer:
Actual time depends too heavily on variables like hardware capabilities, programming languages, and specific compiler implementations. Instead, we count the number of basic operations relative to the input size.

Question 3: Big O Meaning

In asymptotic notation, what does Big O (O) mathematically signify?

🪄 Lumos! Reveal Answer

Answer:
It signifies an asymptotic upper bound (meaning "less than or equal to" the given growth rate).

Question 4: Big Omega Meaning

What does Big Omega (Ω) signify?

🪄 Lumos! Reveal Answer

Answer:
It signifies an asymptotic lower bound (meaning "greater than or equal to" the given growth rate).

Question 5: Big Theta Meaning

What does Big Theta (Θ) signify?

🪄 Lumos! Reveal Answer

Answer:
It signifies an asymptotically tight bound (meaning the function grows exactly at the specified rate).

Question 6: Simplifying Polynomials

When using Big O notation, how do you simplify the expression 3n2 + 20n + 500?

🪄 Lumos! Reveal Answer

Answer:
O(n2). We drop all constants and lower-order terms, keeping only the dominant factor.

Question 7: Order of Growth

Which grows significantly faster as n approaches infinity: O(n3) or O(2n)?

🪄 Lumos! Reveal Answer

Answer:
O(2n). Exponential growth rapidly outpaces polynomial growth.

Question 8: Finding Big O

What is the Big O value for the function f(n) = 2n + n2 + 8n + 7?

🪄 Lumos! Reveal Answer

Answer:
O(2n), because 2n is the highest order term.

Question 9: Basic Operations

An algorithm contains an assignment (s=1), an addition, a multiplication, and a comparison. What is its time complexity?

🪄 Lumos! Reveal Answer

Answer:
O(1), also known as constant time. It does not scale with any input n.

Question 10: Nested Loops

A code fragment has a loop running from 1 to n. Inside it, a second loop runs from 1 to i. What is the Big O complexity?

🪄 Lumos! Reveal Answer

Answer:
O(n2). The inner loop executes a total of roughly n(n+1)/2 times.

Question 11: Recurrence Relation

What is a recurrence relation in the context of algorithms?

🪄 Lumos! Reveal Answer

Answer:
An equation or inequality that describes the running time of a recursive function in terms of its value on smaller inputs.

Question 12: Solving Recurrences

Name the three primary methods taught to solve recurrence relations.

🪄 Lumos! Reveal Answer

Answer:
1. Substitution Method
2. Recursion-tree Method
3. Master Method (Master Theorem)

Question 13: Substitution Method

The Substitution Method requires you to make a mathematical guess. What technique is then used to prove that guess is correct?

🪄 Lumos! Reveal Answer

Answer:
Mathematical Induction.

Question 14: Recursion Tree Method

What is the primary purpose of drawing a Recursion Tree?

🪄 Lumos! Reveal Answer

Answer:
To visualize the cost of subproblems at each level and sum them up, generating a highly accurate guess that can be verified later.

Question 15: Master Theorem Equation

What specific mathematical form must a recurrence take to be solvable using the Master Method?

🪄 Lumos! Reveal Answer

Answer:
T(n) = aT(n/b) + f(n), where a ≥ 1 and b > 1.

Question 16: Master Theorem - The Comparison

When applying the Master Theorem, what two values are you fundamentally comparing against each other?

🪄 Lumos! Reveal Answer

Answer:
You compare the cost function f(n) against the calculated watershed value nlogba.

Question 17: Master Theorem - Case 1

In the Master Theorem, if f(n) is polynomially smaller than nlogba, what is the final solution for T(n)?

🪄 Lumos! Reveal Answer

Answer:
T(n) = Θ(nlogba).

Question 18: Master Theorem - Case 2

In the Master Theorem, if f(n) is exactly equal to nlogba, what is the final solution for T(n)?

🪄 Lumos! Reveal Answer

Answer:
T(n) = Θ(nlogba log n).
