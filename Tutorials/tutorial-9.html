Question 1: Recursive Thinking

What does it fundamentally mean to solve a problem recursively?

🪄 Lumos! Reveal Answer

Answer:
It means finding a way to break the problem down into smaller subproblems, where each subproblem has the exact same form as the original problem.

Question 2: Anatomy of a Recursive Definition

What are the two mandatory steps required to create a valid recursive definition of a function?

🪄 Lumos! Reveal Answer

Answer:
1. The Basis Step
2. The Recursive Step

Question 3: The Basis Step

What is the mathematical purpose of the Basis Step in a recursively defined function?

🪄 Lumos! Reveal Answer

Answer:
It specifies the value of the function at the least possible input integer(s), essentially giving the recursion a point to stop (a base case).

Question 4: The Recursive Step

What is the purpose of the Recursive Step?

🪄 Lumos! Reveal Answer

Answer:
It gives a mathematical rule for finding the function's value at a given integer using its values at smaller integers.

Question 5: Evaluating a Recursive Function (Part 1)

Suppose function f is defined by f(0) = 3 and f(n+1) = 2f(n) + 3. What is the value of f(1)?

🪄 Lumos! Reveal Answer

Answer:
f(1) = 2f(0) + 3 → 2(3) + 3 = 9.

Question 6: Evaluating a Recursive Function (Part 2)

Using the same function f(0) = 3 and f(n+1) = 2f(n) + 3, what is the value of f(2)?

🪄 Lumos! Reveal Answer

Answer:
f(2) = 2f(1) + 3 → 2(9) + 3 = 21.

Question 7: Factorial Basis

What is the strictly defined Basis Step for the recursive factorial function F(n) = n!?

🪄 Lumos! Reveal Answer

Answer:
F(0) = 1.

Question 8: Factorial Recursive Step

What is the Recursive Step formulation for the factorial function F(n+1)?

🪄 Lumos! Reveal Answer

Answer:
F(n+1) = (n+1) * F(n).

Question 9: Fibonacci Initial Conditions

Because the Fibonacci sequence relies on two previous terms, it requires two initial conditions in its Basis Step. What are they?

🪄 Lumos! Reveal Answer

Answer:
f₀ = 0 and f₁ = 1.

Question 10: Fibonacci Recursive Step

What is the exact mathematical formulation for the recursive step of the Fibonacci sequence for n ≥ 2?

🪄 Lumos! Reveal Answer

Answer:
fₙ = fₙ₋₁ + fₙ₋₂.

Question 11: Calculating Fibonacci

Given the definitions above, calculate the specific value of the Fibonacci number f₄.

🪄 Lumos! Reveal Answer

Answer:
f₂ = 1 + 0 = 1.
f₃ = 1 + 1 = 2.
f₄ = f₃ + f₂ = 2 + 1 = 3.

Question 12: Defining Recursive Algorithms

By formal definition, what makes an algorithm a "Recursive Algorithm"?

🪄 Lumos! Reveal Answer

Answer:
An algorithm is recursive if it solves a problem by reducing it to an instance of the exact same problem with a smaller input.

Question 13: Power Algorithm Base Case

In the recursive algorithm for computing aⁿ, what is the base case condition and what does it return?

🪄 Lumos! Reveal Answer

Answer:
If n = 0, the algorithm returns 1.

Question 14: Power Algorithm Recursive Step

In the recursive algorithm for computing aⁿ, what specifically is returned in the else branch?

🪄 Lumos! Reveal Answer

Answer:
It returns a * power(a, n - 1).

Question 15: GCD Mathematical Reduction

The recursive Greatest Common Divisor algorithm is built entirely on what mathematical reduction property?

🪄 Lumos! Reveal Answer

Answer:
gcd(a, b) = gcd(b mod a, a).

Question 16: GCD Base Case

In the recursive GCD algorithm procedure gcd(a, b) where a < b, what triggers the base case, and what is returned?

🪄 Lumos! Reveal Answer

Answer:
If a = 0, it returns b.

Question 17: Binary Search Precondition

What strict condition must be true about the input array/sequence before a Binary Search algorithm can be safely used?

🪄 Lumos! Reveal Answer

Answer:
The sequence must be completely sorted (an increasing sequence of integers).

Question 18: Binary Search Midpoint

In the recursive Binary Search algorithm, how is the midpoint index m mathematically calculated given boundaries i and j?

🪄 Lumos! Reveal Answer

Answer:
m = ⌊(i + j) / 2⌋ (the floor of the average of the two indices).

Question 19: Binary Search Left Branch

In the recursive Binary Search, if the search element x is strictly LESS than the midpoint element aₘ, what recursive call is made?

🪄 Lumos! Reveal Answer

Answer:
It returns binary_search(i, m - 1, x), searching only the left half of the current boundary.

Question 20: Binary Search Failure Case

If the recursive Binary Search exhausts all boundaries and cannot find the element x, what does the algorithm ultimately return?

🪄 Lumos! Reveal Answer

Answer:
It returns 0.
