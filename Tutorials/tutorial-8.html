Question 1: Core Tree Definition

What specifically defines an undirected graph as a Tree?

🪄 Lumos! Reveal Answer

Answer:
It is an undirected graph where there is a unique simple path between any two of its vertices (which means it contains no cycles).

Question 2: Rooted Tree Edges

In a Rooted Tree, how are the edges directed relative to the root?

🪄 Lumos! Reveal Answer

Answer:
Every edge is directed away from the root.

Question 3: Identifying Leaves

What is the definition of a "leaf" in a rooted tree?

🪄 Lumos! Reveal Answer

Answer:
A leaf is a node that has no children.

Question 4: Binary Trees

What makes a rooted tree a "Binary Tree"?

🪄 Lumos! Reveal Answer

Answer:
Every internal vertex has at most two children (a left child and a right child).

Question 5: Complete Binary Trees

What two conditions must be met for a Binary Tree to be considered "Complete"?

🪄 Lumos! Reveal Answer

Answer:
1. Every internal vertex has exactly two children.
2. Absolutely all leaves are at the exact same level.

Question 6: Node Levels

If you are at the root of a tree, what level are you currently on?

🪄 Lumos! Reveal Answer

Answer:
Level 0.

Question 7: Tree Height

How do you find the "height" of a tree?

🪄 Lumos! Reveal Answer

Answer:
The height is the maximum level (or depth) of any node within the tree.

Question 8: Node Degree

In tree terminology, what does the "degree" of a specific node mean?

🪄 Lumos! Reveal Answer

Answer:
It is the number of subtrees originating from that node (the number of its children).

Question 9: Descendants

What nodes make up the descendants of a given node X?

🪄 Lumos! Reveal Answer

Answer:
All the nodes located in the subtrees originating from node X (its children, grandchildren, etc.).

Question 10: Expression Trees - Leaves

When modeling an arithmetic expression as a tree, what do the leaves (external vertices) represent?

🪄 Lumos! Reveal Answer

Answer:
They represent the operands (the numbers or variables).

Question 11: Expression Trees - Internal Nodes

Following the previous question, what do the internal vertices represent?

🪄 Lumos! Reveal Answer

Answer:
They represent the mathematical operators (+, -, *, /).

Question 12: Pre-order Traversal

What is the exact sequence of a Pre-order Traversal?

🪄 Lumos! Reveal Answer

Answer:
Root → Left → Right.

Question 13: Infix Notation Traversal

Which tree traversal method produces standard Infix Notation?

🪄 Lumos! Reveal Answer

Answer:
In-order Traversal (Left → Root → Right).

Question 14: Post-order Traversal

What is the exact sequence of a Post-order Traversal?

🪄 Lumos! Reveal Answer

Answer:
Left → Right → Root.

Question 15: Heaps

What specific type of graph structure must a Heap be built upon?

🪄 Lumos! Reveal Answer

Answer:
A Complete Binary Tree.

Question 16: Max-Heap Property

In a Max-Heap, what is the rule regarding a parent node and its children?

🪄 Lumos! Reveal Answer

Answer:
Every parent node must contain a value greater than or equal to the values of its children.

Question 17: Min-Heap Root

In a Min-Heap, where is the smallest value in the entire structure guaranteed to be?

🪄 Lumos! Reveal Answer

Answer:
It is always located at the root.

Question 18: Spanning Trees

What makes a subgraph a "Spanning Tree" of graph G?

🪄 Lumos! Reveal Answer

Answer:
The subgraph must be a tree, and it must contain absolutely all the vertices of the original graph G.

Question 19: Minimum Spanning Tree

What is a Minimum Spanning Tree (MST)?

🪄 Lumos! Reveal Answer

Answer:
In a weighted graph, it is a spanning tree whose sum of edge weights is as small as possible.

Question 20: Prim's Algorithm Strategy

In Prim's Algorithm, how is the next vertex chosen to be added to the Minimum Spanning Tree?

🪄 Lumos! Reveal Answer

Answer:
It is a greedy choice: it selects a vertex not yet in the tree that is connected by the edge with the absolute minimal weight to a vertex already in the tree.
