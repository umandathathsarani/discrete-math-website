Question 1: Graph Classification

If a graph has multiple edges connecting the exact same pair of vertices, what specific classification of graph is it?

🪄 Lumos! Reveal Answer

Answer:
It is a Multigraph.

Question 2: Generating from Sets

You are given the edge set E = {(a,b), (b,c), (c,a)}. If this is a directed graph, can you traverse from vertex c to vertex b in one step?

🪄 Lumos! Reveal Answer

Answer:
No. In a directed graph, the pair (b,c) means the edge goes strictly from b to c. There is no (c,b) edge listed in the set.

Question 3: Adjacency

If an edge e connects vertex X and vertex Y in an undirected graph, describe the relationship between X and Y.

🪄 Lumos! Reveal Answer

Answer:
Vertex X and Vertex Y are adjacent vertices.

Question 4: Incidence

Following Question 3, what is the relationship between the edge e and vertex X?

🪄 Lumos! Reveal Answer

Answer:
The edge e is incident to vertex X.

Question 5: Vertex Degree

In an undirected graph, a vertex v has 3 standard edges connecting to other vertices, and 1 loop connecting to itself. What is the degree of v?

🪄 Lumos! Reveal Answer

Answer:
The degree is 5. (Standard edges count as 1 each, but a self-loop contributes 2 to the degree of a vertex in an undirected graph).

Question 6: Directed Degrees

In a directed graph, vertex A has edges pointing to B, C, and D. It has edges pointing to it from E and F. What is the indegree and outdegree of A?

🪄 Lumos! Reveal Answer

Answer:
Indegree: 2 (from E, F)
Outdegree: 3 (to B, C, D)

Question 7: Circuits vs Cycles

What is the difference between a simple circuit and a cycle?

🪄 Lumos! Reveal Answer

Answer:
A circuit is any path that starts and ends at the same vertex. A cycle is a strict type of simple circuit where no other vertex is repeated along the way except the start/end point.

Question 8: Hamiltonian Cycles

For a cycle to be considered a "Hamiltonian Cycle", what specific condition must it fulfill?

🪄 Lumos! Reveal Answer

Answer:
It must visit every single vertex in the entire graph exactly once before returning to the start.

Question 9: Adjacency Matrix Values

In an Adjacency Matrix A for an unweighted graph, what do the values of 1 and 0 represent at position aij?

🪄 Lumos! Reveal Answer

Answer:
1 means there is an edge connecting vertex i to vertex j.
0 means there is no edge between them.

Question 10: Matrix Symmetry

Will the Adjacency Matrix of an undirected graph always be symmetrical across its main diagonal? Why?

🪄 Lumos! Reveal Answer

Answer:
Yes. Because if there is an edge between i and j (aij = 1), there is identically an edge between j and i (aji = 1).

Question 11: Identifying Trees

What two conditions must a graph meet to be defined mathematically as a Tree?

🪄 Lumos! Reveal Answer

Answer:
1. It must be connected.
2. It must contain no cycles (acyclic).

Question 12: Forests

If you have an undirected graph that contains no cycles, but it is broken into three completely disconnected pieces, what is it called?

🪄 Lumos! Reveal Answer

Answer:
A Forest (a collection of disconnected trees).

Question 13: Tree Edges Formula

If a tree has exactly 20 vertices, how many edges does it possess?

🪄 Lumos! Reveal Answer

Answer:
19 edges. A tree always has exactly |V| - 1 edges.

Question 14: Spanning Tree Edges

How many edges must be removed from a connected graph with n vertices and m edges to produce a valid spanning tree?

🪄 Lumos! Reveal Answer

Answer:
A spanning tree must have exactly (n - 1) edges. Therefore, you must remove m - (n - 1) edges, which simplifies to m - n + 1.

Question 15: Memory Efficiency

Which graph representation uses less memory for a very large, sparse graph (very few edges): Adjacency Matrix or Adjacency List?

🪄 Lumos! Reveal Answer

Answer:
Adjacency List. It only uses memory proportional to Θ(V + E), whereas a matrix wastes memory storing thousands of zeros, taking O(V2) space.

Question 16: BFS Data Structures

Which fundamental data structure does the Breadth-First Search (BFS) algorithm use to keep track of the vertices it needs to process next?

🪄 Lumos! Reveal Answer

Answer:
A Queue (First-In, First-Out). This ensures it searches level-by-level.

Question 17: DFS Data Structures

Which fundamental data structure does the Depth-First Search (DFS) algorithm use to explore as deep as possible before backtracking?

🪄 Lumos! Reveal Answer

Answer:
A Stack (Last-In, First-Out), which is often implemented implicitly using recursive function calls.

Question 18: BFS Color Meanings

In the BFS algorithm, what does it mean when a vertex is colored GRAY?

🪄 Lumos! Reveal Answer

Answer:
GRAY means the vertex has been discovered and is currently in the queue, but its own neighbors have not all been fully explored yet.

Question 19: DFS Timestamps

In DFS, each vertex receives a timestamp f[v]. What specific event does this timestamp record?

🪄 Lumos! Reveal Answer

Answer:
It records the finish time—the exact moment when the search finishes examining every edge in vertex v's adjacency list (and the vertex is colored BLACK).

Question 20: Algorithm Running Times

What is the total running time for both the BFS and DFS algorithms when implemented using an Adjacency List?

🪄 Lumos! Reveal Answer

Answer:
Both algorithms run in linear time: O(V + E) or Θ(V + E), as they process every vertex and every edge a constant number of times.
