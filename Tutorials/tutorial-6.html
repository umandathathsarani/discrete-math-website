Part 01

Q1: Inequality Proof Structure

For P(n): 2n < (n + 1)!
a. Write P(2). Is it true?
b. Write P(k) and P(k+1).
c. Explain the inductive step.

🪄 Reveal Answer

a) P(2): 4 < 6 (True).
b) P(k): 2k < (k+1)!; P(k+1): 2k+1 < (k+2)!.
c) Assume P(k) is true, then show P(k+1) follows.

Q2: Factorial Inequality

Prove by induction that n! > 2n for n ≥ 4.

🪄 Reveal Answer

Basis: 4! = 24 > 16.
Step: (k+1)! = (k+1)k! > (k+1)2k. Since k ≥ 4, (k+1) > 2, so (k+1)2k > 2k+1.

Q3: Exponential Inequality

Prove by induction: 3n > n2 for n ≥ 1.

🪄 Reveal Answer

Basis: 3 > 1.
Step: Assume 3k > k2. Then 3k+1 = 3(3k) > 3k2. Since 3k2 > (k+1)2 for k≥1, the proof holds.

Q4: Advanced Exponential Inequality

Prove by induction: 3n > n3 for n ≥ 4.

🪄 Reveal Answer

Basis: 81 > 64.
Step: 3k+1 = 3(3k) > 3k3. We need 3k3 > (k+1)3 = k3+3k2+3k+1. This holds for k≥4.

Q5: Divisibility by 3

Prove 22n - 1 is divisible by 3 for n ≥ 0.

🪄 Reveal Answer

Basis: 20 - 1 = 0.
Step: 22(k+1) - 1 = 4(22k) - 1. Substitute 22k = 3m+1. Result: 4(3m+1)-1 = 12m+3, which is 3(4m+1).

Q6: Divisibility by 7

Prove 11n - 4n is divisible by 7.

🪄 Reveal Answer

Basis: 11-4=7.
Step: 11k+1-4k+1 = 11(11k)-4(4k). Replace 11k with 4k+7m. Gives 11(4k+7m)-4(4k) = 7(4k)+77m, divisible by 7.

Q7: Advanced Divisibility

Prove 9n - 8n - 1 is divisible by 64.

🪄 Reveal Answer

Basis: 9-8-1=0.
Step: 9k+1 - 8(k+1) - 1. Replace 9k = 64m+8k+1. Result: 9(64m+8k+1)-8k-9 = 576m+72k+9-8k-9 = 64(9m+k).

Q8: Divisibility by 6

Prove n3 + 5n is divisible by 6.

🪄 Reveal Answer

Basis: 1+5=6.
Step: (k+1)3+5(k+1) = (k3+3k2+3k+1)+5k+5. Regroup: (k3+5k) + 3k(k+1) + 6. Since 3k(k+1) is div by 6, the whole thing is.

Q9: Polynomial Factorization Divisibility

Prove n3 + 3n2 + 2n is divisible by 6.

🪄 Reveal Answer

Basis: 1+3+2=6.
Step: (k+1)3 + 3(k+1)2 + 2(k+1) = (k3+3k2+2k) + 3k2+9k+6. 3k2+9k is 3k(k+3). Matches divisible by 6 logic.

Q10: Divisibility by 4

Prove 5n - 1 is divisible by 4.

🪄 Reveal Answer

Basis: 5-1=4.
Step: 5(5k)-1 = 5(4m+1)-1 = 20m+4 = 4(5m+1).

Q11: Divisibility by 2

Prove 3n - 1 is divisible by 2.

🪄 Reveal Answer

Basis: 3-1=2.
Step: 3(3k)-1 = 3(2m+1)-1 = 6m+2 = 2(3m+1).

Q12: Geometric Series

Prove: ∑i=0n 2i = 2n+1 - 1

🪄 Reveal Answer

Basis: 20 = 1; 21-1 = 1.
Step: (2k+1-1) + 2k+1 = 2(2k+1)-1 = 2k+2-1.

Q13: General Geometric Series

Prove: ∑i=0n ari = a(rn+1 - 1) / (r - 1)

🪄 Reveal Answer

Basis: a=a.
Step: Add ark+1. Factor a/(r-1) and simplify fractions.

Q14: Sum of Squares

Prove: ∑i=1n i2 = n(n+1)(2n+1) / 6

🪄 Reveal Answer

Basis: 1 = 1(2)(3)/6 = 1.
Step: Add (k+1)2 to k(k+1)(2k+1)/6. Factor (k+1)/6 to reach correct formula.

Q15: Sum of Cubes

Prove: ∑i=1n i3 = [n(n+1) / 2]2

🪄 Reveal Answer

Step: [k(k+1)/2]2 + (k+1)3. Factor (k+1)2/4. Result is [(k+1)(k+2)/2]2.

Q16: Telescoping Sum

Prove: ∑i=1n [1 / (i(i+1))] = n / (n+1)

🪄 Reveal Answer

Step: [k / (k+1)] + [1 / ((k+1)(k+2))]. Simplify to (k+1)/(k+2).

Q17: Recurrence Proof

Given an = 2an-1 + 3 with a1 = 5, prove an = 2n+2 - 3.

🪄 Reveal Answer

Basis: 23-3 = 5.
Step: ak+1 = 2(2k+2-3)+3 = 2k+3-6+3 = 2(k+1)+2-3.

Q18: Basic Sequence Proof

Prove ∑i=1n i = n(n+1)/2

🪄 Reveal Answer

Basis: 1 = 1(2)/2.
Step: k(k+1)/2 + (k+1) = (k+1)(k+2)/2.

Q19: Odd Integer Sum

Prove 1+3+5+...+(2n-1) = n2

🪄 Reveal Answer

Basis: 1 = 12.
Step: k2 + (2k+1) = (k+1)2.

Q20: Recursive Product Proof

Prove ∑i=1n i*i! = (n+1)! - 1

🪄 Reveal Answer

Step: Assume ∑ = (k+1)! - 1. Add (k+1)(k+1)!. (k+1)! - 1 + (k+1)(k+1)! = (k+1)!(1 + k+1) - 1 = (k+1)!(k+2) - 1 = (k+2)! - 1.

Part 02

Question 1: Inequality Proof Structure

For each positive integer n, let P(n) be the property 2n < (n + 1)!

  1. Write P(2). Is P(2) true?
  2. Write P(k).
  3. Write P(k+1).
  4. In a proof by mathematical induction that this inequality holds for all integers n ≥ 2, what must be shown in the inductive step?
🪄 Lumos! Reveal Answer

Answers:
a) P(2): 22 < (2+1)! → 4 < 3! → 4 < 6. This is True.
b) P(k): 2k < (k+1)!
c) P(k+1): 2k+1 < (k+2)!
d) We must show that assuming P(k) is true (the inductive hypothesis), P(k+1) must follow.

Question 2: Proving Divisibility

Use mathematical induction to prove that for all integers n ≥ 0, 22n - 1 is divisible by 3.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: Show P(0) is true: 20 - 1 = 1 - 1 = 0, which is divisible by 3.
Inductive Step: Assume 22k - 1 = 3m for some integer m. Show 22(k+1) - 1 is also divisible by 3.
Hint: 22k+2 - 1 = 4(22k) - 1 = 4(22k - 1 + 1) - 1 = 4(22k - 1) + 3, which is clearly divisible by 3.

Question 3: Proving Summation Formulas

Use mathematical induction to prove that ∑j=0n arj = (arn+1 - a) / (r - 1) where r ≠ 1 and n is a nonnegative integer.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(0) → ar0 = a(r1 - a) / (r - 1) → a = a(r - 1)/(r - 1) → a = a. (True).
Inductive Step: Assume ∑j=0k arj = (ark+1 - a) / (r - 1). Show it for k+1.
Add the k+1 term: (ark+1 - a)/(r - 1) + ark+1. Simplify to match the formula for n=k+1.

Question 4: Sum of Squares

Use induction to prove the formula for the sum of the first n squares: ∑i=1n i2 = n(n+1)(2n+1) / 6

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 12 = 1(1+1)(2(1)+1) / 6 = 6/6 = 1. (True).
Inductive Step: Assume true for P(k). Add the (k+1)2 term to the sum: [k(k+1)(2k+1)/6] + (k+1)2.
Factor out (k+1)/6 to get: [(k+1)/6] * [k(2k+1) + 6(k+1)] = [(k+1)/6] * [2k2 + 7k + 6] = [(k+1)(k+2)(2k+3)/6].

Question 5: Power of 7 Divisibility

Use mathematical induction to prove that 7n - 1 is divisible by 6 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 71 - 1 = 6, which is divisible by 6. (True).
Inductive Step: Assume 7k - 1 = 6m (where m is an integer).
Show 7k+1 - 1 is divisible by 6: 7(7k) - 1 = 7(6m + 1) - 1 = 42m + 7 - 1 = 42m + 6 = 6(7m + 1).
Since the result is a multiple of 6, it is divisible by 6.

Question 6: Geometric Series

Use induction to prove: ∑i=1n 2i-1 = 2n - 1

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 20 = 21 - 1 → 1 = 1. (True).
Inductive Step: Assume ∑i=1k 2i-1 = 2k - 1.
For P(k+1), add the next term 2k: (2k - 1) + 2k = 2(2k) - 1 = 2k+1 - 1.

Question 7: Factorial Inequality

Prove by induction that n! > 2n for all integers n ≥ 4.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(4) → 4! = 24 and 24 = 16. 24 > 16. (True).
Inductive Step: Assume k! > 2k.
For P(k+1): (k+1)! = (k+1) * k! > (k+1) * 2k.
Since k ≥ 4, (k+1) is at least 5. Therefore, 5 * 2k > 2 * 2k = 2k+1.

Question 8: Power of 3 Divisibility

Prove by induction that 3n - 1 is divisible by 2 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 31 - 1 = 2, which is divisible by 2. (True).
Inductive Step: Assume 3k - 1 = 2m.
For P(k+1): 3k+1 - 1 = 3(3k) - 1. Substitute 3k = 2m + 1:
3(2m + 1) - 1 = 6m + 3 - 1 = 6m + 2 = 2(3m + 1), which is divisible by 2.

Question 9: Sum of Cubes

Use induction to prove the formula for the sum of cubes: ∑i=1n i3 = [n(n+1) / 2]2

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 13 = [1(2)/2]2 = 1. (True).
Inductive Step: Assume true for P(k). Add (k+1)3 to the sum: [k(k+1)/2]2 + (k+1)3.
Factor out (k+1)2: [(k+1)2 / 4] * [k2 + 4(k+1)] = [(k+1)2 / 4] * [k2 + 4k + 4] = [(k+1)2(k+2)2 / 4]. This matches the formula for n=k+1.

Question 10: Divisibility by 4

Use mathematical induction to prove that 5n - 1 is divisible by 4 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 51 - 1 = 4, which is divisible by 4. (True).
Inductive Step: Assume 5k - 1 = 4m.
For P(k+1): 5k+1 - 1 = 5(5k) - 1. Substitute 5k = 4m + 1:
5(4m + 1) - 1 = 20m + 5 - 1 = 20m + 4 = 4(5m + 1), which is divisible by 4.

Question 11: Inequality Proof

Prove by induction that 3n > n2 for all integers n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 31 > 12 → 3 > 1. (True).
Inductive Step: Assume 3k > k2.
For P(k+1): 3k+1 = 3 * 3k > 3k2.
We need to show 3k2 > (k+1)2 = k2 + 2k + 1.
This simplifies to 2k2 - 2k - 1 > 0, which holds true for all k ≥ 2. (The base cases n=1 and n=2 can be checked individually to satisfy the range).

Question 12: Divisibility by 7

Use induction to prove that 11n - 4n is divisible by 7 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 11 - 4 = 7, which is divisible by 7. (True).
Inductive Step: Assume 11k - 4k = 7m → 11k = 4k + 7m.
For P(k+1): 11k+1 - 4k+1 = 11(11k) - 4(4k).
Substitute: 11(4k + 7m) - 4(4k) = 11(4k) + 77m - 4(4k) = 7(4k) + 77m = 7(4k + 11m). This is divisible by 7.

Question 13: Closed Form Verification

Given the recurrence relation an = 2an-1 + 1 with a1 = 1, use induction to prove the closed form is an = 2n - 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → a1 = 21 - 1 = 1. (True).
Inductive Step: Assume ak = 2k - 1.
For P(k+1): ak+1 = 2(ak) + 1 = 2(2k - 1) + 1 = 2k+1 - 2 + 1 = 2k+1 - 1. (True).

Question 14: Divisibility by 6

Use induction to prove that n3 + 5n is divisible by 6 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(1) → 13 + 5(1) = 6, divisible by 6. (True).
Inductive Step: Assume k3 + 5k = 6m.
For P(k+1): (k+1)3 + 5(k+1) = (k3 + 3k2 + 3k + 1) + 5k + 5.
Rearrange: (k3 + 5k) + 3k2 + 3k + 6 = (6m) + 3k(k+1) + 6.
Since k(k+1) is always even (product of two consecutive integers), 3k(k+1) is divisible by 6. The sum is divisible by 6.

Question 15: Sum of Powers of 2

Prove by induction that ∑i=0n 2i = 2n+1 - 1

🪄 Lumos! Reveal Answer

Answer Structure:
Basis Step: P(0) → 20 = 1 and 20+1 - 1 = 1. (True).
Inductive Step: Assume ∑i=0k 2i = 2k+1 - 1.
For P(k+1): Add the next term 2k+1 to the sum: (2k+1 - 1) + 2k+1 = 2 * (2k+1) - 1 = 2k+2 - 1. (True).

Question 16: Telescoping Sum

Use induction to prove: ∑i=1n [1 / (i(i+1))] = n / (n+1)

🪄 Lumos! Reveal Answer

Step-by-Step Solution:
Basis Step: For n=1: 1/(1*2) = 1/2. Formula: 1/(1+1) = 1/2. Matches.
Inductive Step: Assume ∑i=1k [1 / (i(i+1))] = k / (k+1).
For P(k+1), add the next term 1/((k+1)(k+2)):
[k / (k+1)] + [1 / ((k+1)(k+2))]
Factor out 1/(k+1): [1/(k+1)] * [k + 1/(k+2)]
= [1/(k+1)] * [(k(k+2) + 1) / (k+2)]
= [1/(k+1)] * [(k2 + 2k + 1) / (k+2)]
= [1/(k+1)] * [(k+1)2 / (k+2)] = (k+1) / (k+2). Matches.

Question 17: Polynomial Factorization

Prove by induction that n3 + 3n2 + 2n is divisible by 6 for all n ≥ 1.

🪄 Lumos! Reveal Answer

Step-by-Step Solution:
Basis Step: n=1: 1 + 3 + 2 = 6. Divisible by 6. (True).
Inductive Step: Assume k3 + 3k2 + 2k = 6m.
For P(k+1): (k+1)3 + 3(k+1)2 + 2(k+1)
Expand: (k3 + 3k2 + 3k + 1) + 3(k2 + 2k + 1) + 2k + 2
Group the k-terms: (k3 + 3k2 + 2k) + 3k2 + 3k + 3 + 3k + 3
= (6m) + 3k2 + 9k + 6
Note that 3k2 + 3k = 3k(k+1). Since k(k+1) is always even, 3k(k+1) is divisible by 6.
Therefore, 6m + 6 + (multiple of 6) is divisible by 6.

Question 18: Recurrence Relation Proof

A sequence is defined as an = 2an-1 + 3 with a1 = 5. Prove by induction that the closed form is an = 2n+2 - 3.

🪄 Lumos! Reveal Answer

Step-by-Step Solution:
Basis Step: n=1: 21+2 - 3 = 23 - 3 = 8 - 3 = 5. Matches a1.
Inductive Step: Assume ak = 2k+2 - 3.
For P(k+1): ak+1 = 2(ak) + 3
= 2(2k+2 - 3) + 3
= 2k+3 - 6 + 3
= 2(k+1)+2 - 3. Matches.

Question 19: Inequality with Exponential

Use induction to prove that 3n > n3 for all integers n ≥ 4.

🪄 Lumos! Reveal Answer

Step-by-Step Solution:
Basis Step: n=4: 34 = 81 and 43 = 64. 81 > 64. (True).
Inductive Step: Assume 3k > k3.
For P(k+1): 3k+1 = 3 * 3k > 3k3.
We need 3k3 > (k+1)3 = k3 + 3k2 + 3k + 1.
This requires 2k3 - 3k2 - 3k - 1 > 0. For k=4, 2(64) - 3(16) - 12 - 1 = 128 - 48 - 13 = 67 > 0. The function grows for k > 4.
