Question 1: The Core Problem

What is the primary objective of "String Matching"?

🪄 Lumos! Reveal Answer

Answer:
Finding one or more occurrences of a specific pattern string within another string or body of text.

Question 2: Applications

Name at least three real-world applications of string matching algorithms.

🪄 Lumos! Reveal Answer

Answer:
1. Text editing (Find and Change)
2. Computer security (Virus detection, password checking)
3. DNA sequence analysis
4. Data communications (Header analysis)

Question 3: Automaton Efficiency

When using a finite automaton for string matching, how many times does the algorithm examine each text character?

🪄 Lumos! Reveal Answer

Answer:
It examines each text character exactly once.

Question 4: Time Complexity

Because an automaton scans each character exactly once, what is the time complexity required per text character?

🪄 Lumos! Reveal Answer

Answer:
It takes constant time (O(1)) per text character.

Question 5: The Automaton Tuple

A finite automaton M is formally defined as a mathematical tuple containing how many elements?

🪄 Lumos! Reveal Answer

Answer:
It is a 5-tuple: (Q, q0, A, Σ, δ).

Question 6: Definition of Q

In the 5-tuple, what does Q represent?

🪄 Lumos! Reveal Answer

Answer:
A finite set of states.

Question 7: Definition of q0

In the 5-tuple, what does q0 represent?

🪄 Lumos! Reveal Answer

Answer:
The start state of the automaton.

Question 8: Definition of A

In the 5-tuple, what does A represent?

🪄 Lumos! Reveal Answer

Answer:
The distinguished set of accepting states (which is a subset of Q).

Question 9: Definition of Σ

In the 5-tuple, what does the Greek letter Sigma (Σ) represent?

🪄 Lumos! Reveal Answer

Answer:
The finite input alphabet (e.g., the characters {a, b}).

Question 10: The Transition Function

The transition function δ (Delta) maps two inputs to one output. What are the two inputs?

🪄 Lumos! Reveal Answer

Answer:
It maps the current state (from Q) and the current input character (from Σ) to determine the next state.

Question 11: Diagram Edges

In a state-transition diagram, what do the directed arrows (edges) represent?

🪄 Lumos! Reveal Answer

Answer:
They represent the state transitions (the execution of the δ function based on an input character).

Question 12: Accepting States Visual

In the lecture's state-transition diagrams, how is an accepting state visually differentiated from a normal state?

🪄 Lumos! Reveal Answer

Answer:
Accepting states are typically blackened or shaded (and commonly drawn with a double circle in other notations).

Question 13: Tracing Inputs (Part 1)

An automaton accepts strings ending in an odd number of 'a's. State 0 is Start, State 1 is Accept. Tracing the input abaaa results in sequence (0, 1, 0, 1, 0, 1). Is this accepted?

🪄 Lumos! Reveal Answer

Answer:
Yes, because the final state is 1, which is the accepting state.

Question 14: Tracing Inputs (Part 2)

Tracing the input abbaa results in the sequence (0, 1, 0, 0, 1, 0). Is this accepted?

🪄 Lumos! Reveal Answer

Answer:
No, because the final state is 0, which is a rejecting state.

Question 15: The "Spine" of the Diagram

When looking at a complex automaton diagram built for a specific pattern (like "ababaca"), what do the right-going edges (the "spine") represent?

🪄 Lumos! Reveal Answer

Answer:
They represent successful matches between the pattern and the text characters.

Question 16: Left-going Edges

Conversely, what do the left-going edges in a pattern-matching automaton represent?

🪄 Lumos! Reveal Answer

Answer:
They represent failing matches, causing the automaton to fall back to an earlier state to continue scanning.

Question 17: Implicit Edges

By standard convention, if a state i has no explicitly drawn outgoing edge for a character a, what is the assumed value of δ(i, a)?

🪄 Lumos! Reveal Answer

Answer:
It is assumed to be 0 (it resets to the start state).

Question 18: DNA Alphabet

If you were building a finite automaton specifically for DNA sequence analysis, what would your input alphabet Σ likely be?

🪄 Lumos! Reveal Answer

Answer:
Σ = {A, C, G, T}.

Question 19: Transition Tables

In a tabular representation of a transition function, what do the rows, columns, and internal cells represent?

🪄 Lumos! Reveal Answer

Answer:
Rows represent the current state, columns represent the input character, and the cell value is the next state.

Question 20: Finding the Match

In the "ababaca" example, if φ(T9) equals 7, and 7 is the only accepting state, what has just occurred?

🪄 Lumos! Reveal Answer

Answer:
One full, successful occurrence of the pattern was found, ending exactly at text position 9.
