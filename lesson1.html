Lecture 1: Set Theory

1. Definitions and Terminology

A Set is an unordered, well-defined collection of distinct objects. The objects in a set are called its elements or members.

  • x ∈ A: x is an element of set A.
  • x ∉ A: x is not an element of set A.
  • Order does not matter: {1, 2, 3} is the same as {3, 1, 2}.
  • Repetitions are ignored: {1, 1, 2, 3, 3} is exactly the same set as {1, 2, 3}.

2. Representing Sets

There are two primary ways to describe a set:

  • Roster Method: Listing all the elements explicitly inside braces.
    Example: V = {a, e, i, o, u}
  • Set-Builder Notation: Stating the properties that all elements must satisfy.
    Example: O = {x | x is an odd positive integer less than 10}

3. Standard Sets of Numbers

Important mathematical sets have standard symbols:

  • ℕ (Natural Numbers): {0, 1, 2, 3, ...}
  • ℤ (Integers): {..., -2, -1, 0, 1, 2, ...}
  • ℤ⁺ (Positive Integers): {1, 2, 3, ...}
  • ℚ (Rational Numbers): {p/q | p ∈ ℤ, q ∈ ℤ, and q ≠ 0}
  • ℝ (Real Numbers): All rational and irrational numbers.
  • ℂ (Complex Numbers): Numbers in the form a + bi.

4. Types of Sets & Cardinality

  • Empty Set (Null Set): A set with no elements. Denoted by ∅ or {}.
  • Singleton Set: A set containing exactly one element. Example: {a}.
  • Universal Set (U): The set containing everything currently under consideration.
  • Cardinality (|S|): The number of distinct elements in a set S. If S = {1, 2, 3}, then |S| = 3.

5. Subsets and Power Sets

  • Subset (A ⊆ B): A is a subset of B if every element of A is also in B.
  • Proper Subset (A ⊂ B): A is a subset of B, but A is not perfectly equal to B (B has at least one extra element).
  • Power Set (𝒫(A)): The set of all possible subsets of A, including the empty set and A itself.
    Rule: If a set A has n elements, its Power Set will have 2n elements.

6. Set Operations

Key operations used to combine or modify sets:

  • Union (A ∪ B): Elements that belong to A, or B, or both. {x | x ∈ A ∨ x ∈ B}
  • Intersection (A ∩ B): Elements that belong to both A and B. {x | x ∈ A ∧ x ∈ B}
  • Disjoint Sets: Two sets are disjoint if their intersection is the empty set (A ∩ B = ∅).
  • Difference (A - B): Elements belonging to A but NOT to B. {x | x ∈ A ∧ x ∉ B}
  • Complement (Ac or A'): All elements in the Universal set U that are NOT in A. {x ∈ U | x ∉ A}

7. Cartesian Product

The Cartesian Product of sets A and B, denoted by A × B, is the set of all ordered pairs (a, b) where a ∈ A and b ∈ B.

A × B = {(a, b) | a ∈ A ∧ b ∈ B}

Note: A × B is generally NOT equal to B × A.

8. Important Set Identities (Laws)

These logical rules are used to simplify complex set expressions:

  • Identity Laws:
    A ∪ ∅ = A
    A ∩ U = A
  • Domination Laws:
    A ∪ U = U
    A ∩ ∅ = ∅
  • Idempotent Laws:
    A ∪ A = A
    A ∩ A = A
  • Complementation Law:
    (Ac)c = A
  • Commutative Laws:
    A ∪ B = B ∪ A
    A ∩ B = B ∩ A
  • Associative Laws:
    A ∪ (B ∪ C) = (A ∪ B) ∪ C
    A ∩ (B ∩ C) = (A ∩ B) ∩ C
  • Distributive Laws:
    A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
    A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • De Morgan's Laws:
    (A ∪ B)c = Ac ∩ Bc
    (A ∩ B)c = Ac ∪ Bc
  • Absorption Laws:
    A ∪ (A ∩ B) = A
    A ∩ (A ∪ B) = A
  • Complement Laws:
    A ∪ Ac = U
    A ∩ Ac = ∅
