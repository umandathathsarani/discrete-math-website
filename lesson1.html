Lecture 1: Set Theory

1. Definitions and Notations

A Set is any well-defined collection of objects. The elements of a set are the objects within it.

  • x ∈ A: means x is a member of set A.
  • x ∉ A: means x is not a member of set A.

2. Set Operations

Key operations used to combine or modify sets:

  • Union (A ∪ B): Elements in A, B, or both.
  • Intersection (A ∩ B): Elements in both A and B.
  • Difference (A - B): Elements in A but not in B.
  • Complement (Ac): Elements in the universal set U but not in A.

3. Important Identities

De Morgan's Laws:

  • (A ∪ B)c = Ac ∩ Bc
  • (A ∩ B)c = Ac ∪ Bc
