🎯 Module Aim

The objective of this unit is to provide an understanding of the mathematical concepts required to follow computing units. The concepts discussed in this unit include basic concepts in Number Systems, Boolean Algebra, analysis of algorithms, set theory, finite state machine and some topics in discrete mathematics.

🧠 Learning Outcomes

  • LO1: Appy set theory function and sequence to solve real-world problems
  • LO2: Apply counting principles to solve problems in real world
  • LO3: Explain essential concepts in graph theory and related algorithms
  • LO4: Evaluate and the recursive algorithms and develop recurrence
  • LO5: Analyze the algorithms based on Big O notation to solve complex problems

📚 Contents of the Module

  • 1. Introduction to Sets (12 hours): Operation on Sets, Venn Diagrams, Power sets
  • 2. Functions and Sequence (15 hours): Definition of a Function, One to one and onto functions, Inverse and Composite function, Introduction to sequence and summation
  • 3. Graph theory (11 hours): Directed and undirected graphs, paths, cycles, trees, Eulerian cycles, matchings and coverings, connectivity, BFS and DFS
  • 4. Counting (11 hours): Permutation, combination and Binomial Theorem, Pascal triangle
  • 5. Recurrence Relations and mathematical induction (11 hours): Introduction to Recursion, Solve the recursive equation using repeated substitution, Recursion tree, Master theorem, Mathematical Induction
  • 6. Analysis of Algorithms (10 hours): Introduction to algorithms, Big Notation, Time complexity analysis, RAM model, Recursive Algorithm and time complexity analysis with recurrence
