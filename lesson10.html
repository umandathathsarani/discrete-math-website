Lecture 10: Analysis of Algorithms

1. Algorithm Complexity

We analyze algorithms to predict performance independent of hardware, counting basic operations instead of seconds.

  • Time Complexity: Running time as a function of input size.
  • Space Complexity: Memory required as a function of input size.
  • Asymptotic Bounds: Big-O (Upper bound), Big-Omega (Lower bound), Big-Theta (Tight bound).

2. The Master Theorem

Solves recurrence relations of the form: T(n) = aT(n/b) + f(n)

Compares f(n) to nlogba to determine the tight asymptotic bound across three distinct cases.
