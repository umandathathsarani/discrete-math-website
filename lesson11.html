Lecture 11: Finite Automata & String Matching

1. Finite Automaton (M)

A machine model used for string matching and text processing (e.g., DNA analysis, virus detection). It scans text strictly in constant time per character.

Defined as a 5-tuple: (Q, q₀, A, Σ, δ)

  • Q: Finite set of states.
  • q₀: Start state.
  • A: Set of accepting states.
  • Σ: Input alphabet.
  • δ: Transition function mapping a state and input character to a new state.
