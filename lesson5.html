Lecture 5: Combinations & Binomial Theorem

1. What are Combinations?

A combination is an unordered arrangement of objects. Unlike permutations, the order in which objects are arranged does not matter.

2. Combinations without Repetition

If we select r objects from a set of n distinct objects, the number of r-combinations is given by:

C(n, r) = n! ⁄ r! × (n - r)!

3. Combinations with Repetition

If items can be chosen multiple times, the number of r-combinations from a set of n elements is:

C(n + r - 1, r) = (n + r - 1)! ⁄ r! × (n - 1)!

4. Binomial Theorem

The Binomial Theorem describes how to expand expressions of the form (x + y)n. The expansion is given by:

(x + y)n = Σk=0 to n C(n, k) xn-k yk

Pascal's Identity is a way to calculate binomial coefficients, which forms Pascal's Triangle, where each number is the sum of the two numbers directly above it.
