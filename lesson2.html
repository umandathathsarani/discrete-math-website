Lecture 2: Functions

1. Definition and Terminology

A function f from a set X to a set Y relates each element of X to a unique element of Y.

  • Domain: The set of all possible input values (X).
  • Codomain: The set from which the function's output values are chosen (Y).
  • Range (Image): The actual set of all values taken by the function (a subset of the Codomain).

2. Types of Functions

  • One-to-One (Injective): No two different elements in the domain map to the same element in the codomain. If f(a) = f(b), then a = b.
  • Onto (Surjective): Every element in the codomain is mapped to by at least one element in the domain. (Range = Codomain).
  • Bijective: A function that is both One-to-One and Onto.

3. Inverse and Composition

Inverse Function (f⁻¹): If f is bijective, it has an inverse. If f(x) = y, then f⁻¹(y) = x.

Composition of Functions (f ∘ g): The process of applying one function to the results of another. (f ∘ g)(x) = f(g(x)).
